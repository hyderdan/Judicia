# JUDICIA

## Introduction

it is an online judicial management system that allows users to file cases, submit digital evidence, and monitor case progress remotely

- [x] Notification System



## Dependencies

## Backend(FastAPI) – High-performance Python backend framework
Uvicorn – ASGI server for running FastAPI
SQLite – Lightweight relational database
SQLAlchemy – ORM for database management
Dotenv: For loading environment variables from a .env file.

## Frontend (Angular)
React:– Component-based UI development
Tailwind CSS – Utility-first responsive styling
Jwt-Decode (if used): For decoding JWT 
Fetch API / Axios – For backend communication

## AI Model
Google Gemini API – AI-powered media authenticity detection
🖼️ Image Processing
Pillow (PIL) – Image loading and processing
🎥 Video Processing
OpenCV – Extracting frames from video for AI analysis
